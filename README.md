# Тестовое задание (инструкция запуска)

Для запуска тестового задания нужно выполнить следующие шаги:

1.  Скопируйте репозиторий на свой компьютер и перейдите в него:
    ```bash
    git clone git@github.com:shachenkovan/test_task.git
    cd test_task
    ```

2. Создайте .env с переменными окружения:
    ```bash
    touch .env
    ```

3. Вставьте в этот файл следующее содержимое:
    ```
    DB_HOST=localhost (или ваш хост)
    DB_PORT=5432
    DB_USER=<ваш никнейм>
    DB_PASS=<ваш пароль>
    DB_NAME=test_task
   ```

4. Запустите приложение с помощью команды:
    ```bash
   docker-compose up --build
   ```
   
5. Протестировать API можно по ссылке:
    ```
   http://localhost:8000/docs
   ```
